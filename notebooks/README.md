# Portfolio Analytics - Educational Notebooks

This directory contains interactive Jupyter notebooks for learning quantitative finance concepts.

## Structure

- `01_introduction/` - Getting started and market data setup
- `02_option_pricing/` - Black-Scholes-Merton, Binomial Trees, Monte Carlo, PDE methods
- `03_greeks/` - Analytical and numerical Greeks calculation
- `04_volatility/` - Implied volatility, volatility smile, SABR model
- `05_stochastic_processes/` - GBM, Jump Diffusion, Heston model
- `06_exotic_options/` - Barrier options, Asian options
- `07_risk_management/` - Delta hedging, PnL attribution
- `utils/` - Helper functions for notebooks

## Installation

Install the notebooks dependencies:

```bash
pip install portfolio_analytics[notebooks]
```

## Usage

Start Jupyter:

```bash
jupyter notebook
```

Or use JupyterLab:

```bash
jupyter lab
```
